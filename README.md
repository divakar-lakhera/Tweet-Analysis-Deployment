# How to test backend:
> Clone the repo 

    https://github.com/divakar-lakhera/Tweet-Analysis-Deployment.git
    
    
> Using terminal 

    $ cd Backend
    $ pip3 install -r requirements.txt
    $ python3 wsgi.py
    
 # How to test frontend:
> Clone the repo 

    https://github.com/divakar-lakhera/Tweet-Analysis-Deployment.git
    
    
> Using terminal 

    $ cd Frontend/ml-frontend/
    $ npm run serve        [you may need to initialise Vue JS]
